?遗失的童真


在我们对未来生活充满美好向往的时候，被告知世界是物质的世界，被告知如今就业压力剧增，被告知读书是惟一的出路。

在我们用新奇的眼光期待新事物的时候，被告知世界是现实的世界，被告知现实里面没有奇迹，被告知货币才是一切幸福。

当我们真诚友善对待他人，却得到官方式的微笑和感谢。

当我们真心真意对待朋友，却得到潜规则的情感和宣泄。

当我们心驰疆野满怀憧憬，却被告知暮霭沉沉楚天阔。

以前，我们怀着交朋友的心情上飞信、上QQ，希望可以得到更多的交流。

现在，我们仅怀着等待一个人上线的心情，其它多余的交流，貌似不再重要。
