梦之都：不熄灭的权力之火


和人对骂、加友被删、加群被踢、账号被封、发帖被喷，有木有！

梦之都强烈反对类似行为！ 网络是充满舆论的地方，是一个娱乐、发泄的场所，你可以善良，但别人不会，因为网络的身份是虚假的，地域又相差很远，既不知道对方是谁，又没人知道自己是谁。

很多人知道“翔”字的由来，网友“军神李翔”独战群舌，最后落了个无奈的收场。其实这种事情的发生，处于弱势的人是不会希望发生的。这是一个网络社交失败的典型，而且这种事情就发生在商业化比较重的百度贴吧里。百度贴吧的管理员活不见人，吧主更是不知道是什么来历，是不是好人还不一定。

还有关于央视名嘴崔永元的网络战，论证关于转基因的问题。并不清楚其中的内涵，只是妄加猜想而已，网友们有一搭没一搭地冷嘲热讽，激怒了崔永元，引发其赴美国拍纪录片的行动。算不算是又一个网络社交的例子呢？网络舆论的力量是可怕的。

很多人都有跟风的习惯，人云亦云，听说什么东西好，买什么，听说哪个地方好玩，去哪里。网络上更是一样，和现实比起来，网络上的更多东西都是普通网民不知道、不懂或是无法评测的。一部电影，可能有一个人说不好，罗列出电影的一些问题，大家都会觉得不好。但是，从艺术的角度讲，不言而喻。

在跟风的“习俗”下，一篇帖子，有一个人喷你，就有可能招来更多种种的恶语。

虎绿林曾经有一个站长，写了一个导航程序，发帖让人试一下，结果有一个人填写的导航内容都是“站长sb”字样，站长很生气，但是，删也不是，不删也不是，删的话难道就默默咽下这口气？不删的话总不能一直留着吧？

这种低素质发起者的人身攻击，不新鲜，也没意思，损人不利己，但对于当事者，无缘无故遇到这样的事情，尤其是对于接触人事少的学生来说，具有很大杀伤力。


梦之都坚决反对恶意行为！

那么，怎么反对？用什么来保证“反对”的执行？网站管理权限。

要改善网络环境，不是让坏人变好，而是提高门槛，把有恶意的人拒之门外。

如果遇到不合理的事情，可以反映给网站管理。

如果发生个人争执，可以采用公开征集意见的方式解决。

任何一位没有恶意的会员，都可以申请网站管理权限。


梦之都发展现状

很高兴，前天的昨天，8月6日，梦之都迎来第一位注册会员，是昨天在后台无意看到的。

第一位会员，是一个里程碑，尽管有很多的尽管。


网站成立一个月左右，尽管无人问津，但自己是很喜欢网站的，尤其是像discuz这样功能齐全的论坛程序，目前虽然只有一个人注册，而且还是英文名，很英式化得英文名，用的还是gmail谷歌邮箱，也就是说，他很有可能不是中国人，很有可能永远不会再来访问第二次，但是，很高兴他的到来。 360旗下搜索好搜收录网站40多个页面，在自己的建站历史上也是没有的，尽管，2位数的收录数量实在是微不足道，尽管，排名可能在5位数以上了。


电脑用户，上qq，玩游戏，听音乐，看直播。浏览sex内容，看小说，还有什么事情做的吗？很难想象，会打开浏览器进入一个陌生的网站，况且，梦之都的流量来路特别少。

手机用户，上qq，玩游戏，听音乐，看直播，浏览sex内容，看小说……

还有看电影、看电视。

但互联网真正强大的地方可能不是那些媒体内容。那些东西都被大型公司垄断。

网络的范围很广泛，大多数人使用网络的范围却很狭隘。
