那要不要考研？

考研大概有这么几种原因：

1.爱学习，爱生活，为知识而拼搏

    这样的人高考成绩一定炸，已经与我无关

2.认为高学历就能够找到好工作

    考研：

        成本：3年的读研时间

              1年以上复习、准备考研的时间，大量精力

        回报：研究生学历

              认识更多厉害的人，包括导师

    就业:

        成本：3年的工作时间

        回报：工作经历

              薪水

    老梁的观点是，研究生无用，因为目前大多数基层工作岗位需要的技能，本科学历足够。

    读研，企业需要付给你更高的薪水。但是，你能带来比本科学历的人更高的价值吗？

    研究生三年，国内学校的尿性是人尽皆知的，尤其是实战操作能力，三年不一定能够学到太多东西，反而本科毕业有整整三年工作经历的人，工作能力上的提高，不见得比三年学院生活少。

    卿未离，知乎上最近在关注的很厉害的一个人，他对读研的观点是，如果想要在一线城市北上广深从事技术性强的相关工作，就可以选择读研，否则，该干嘛干嘛。

    也就是说，只要不在一线城市，无论从事什么工作，都不要读研。

    当然，事业单位除外，什么公务员、老师，甚至从政。

    张雪峰是考研神嘴，提倡考研，比较关键的一个观点是，考研，可以让你认识一个非常牛的人物，那就是你的导师，甚至如果你将来的就业可以用上那么一点他的人脉资源，那就赚大了。
